# Exercícios de JavaScript

Este repositório contém exercícios de JavaScript organizados por temas e aulas. Cada pasta representa um conjunto de tarefas ou exemplos práticos com foco em conceitos específicos.

## Estrutura do repositório

- `02 conseito boolean e condicionais/`
  - Exercícios sobre valores booleanos e estruturas condicionais.
  - Arquivo principal: `aula.js`, `desafio.js`.

- `03 medias de notas/`
  - Cálculo de médias e lógica relacionada a notas.
  - Arquivo principal: `media.js`.

- `04 IMC/`
  - Exercício de cálculo de IMC (Índice de Massa Corporal).
  - Arquivo principal: `imc.js`.

- `07 Exemplo Prático com Funções/`
  - Exemplo prático demonstrando o uso de funções em JavaScript.
  - Arquivo principal: `aula.js`, `index.html`.

- `10 Estrutura Básica de Um Objeto/`
  - Introdução à criação de objetos em JavaScript.
  - Arquivo principal: `aula.js`.

- `5 sayMyName/`
  - Exercícios com função de apresentação e manipulação de nomes.
  - Arquivos: `aula.js`, `index.html`.

- `6 Como Organizar as Funções/`
  - Organização de funções e melhores práticas no código.
  - Arquivos: `aula.js`, `index.html`.

- `8 Exercício Guiado 1 - Função escrevaMeuNome/`
  - Exercício guiado para criar uma função que escreve o nome.
  - Arquivos: `aula.js`, `index.html`.

- `9 Exercício Guiado 4 - Funções Para Calcular Preço/`
  - Funções voltadas ao cálculo de preço e aplicações práticas.
  - Arquivos: `aula.js`, `index.html`.

- `aula 1/`
  - Conteúdo da primeira aula, com exemplos básicos de JavaScript.
  - Arquivos: `aula.js`, `README.md`.

## Como usar

1. Abra a pasta do exercício desejado no seu editor.
2. Verifique os arquivos `aula.js` e, quando presente, `index.html`.
3. Execute ou teste os exemplos diretamente no navegador (para projetos com `index.html`) ou usando um ambiente Node.js.

## Observações

- A numeração das pastas segue a ordem de aprendizado dos exercícios.
- Algumas pastas são exemplos práticos com HTML para rodar diretamente no navegador.
- Use este README como ponto de partida para navegar pelos exercícios.
